Resumo do Projeto

A FutureWork Network é uma aplicação SPA desenvolvida em React + Vite + Tailwind CSS, criada para simular uma rede profissional voltada ao futuro do trabalho.
O objetivo é conectar talentos de diferentes áreas, permitindo explorar competências técnicas, comportamentais e trajetórias profissionais alinhadas às novas demandas do mercado.

A plataforma apresenta:

Listagem de profissionais fictícios armazenados em JSON local (60 perfis).

Cards com informações essenciais.

Modal interativo com dados completos do profissional.

Sistema de busca e filtros (área, cidade e tecnologia).

Ações funcionais:

Recomendar profissional

Enviar mensagem

Dark Mode com alternância em tempo real.

Design responsivo, limpo e consistente.

O projeto atende integralmente o que foi solicitado na GS, incluindo interface moderna, interatividade, manipulação de estados e organização de componentes.
